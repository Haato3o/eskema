### Eskema - Types

Eskema supports many primitive types as well as some more specific types. The supported types are:

- `String`
- `Char`
- `UInt8`
- `UInt16`
- `UInt32`
- `UInt64`
- `Int8`
- `Int16`
- `Int32`
- `Int64`
- `Float`
- `Double`
- `TimeStamp`
- `Date`
- `DateTime`
- `Array<TValue>`
- `Map<TKey, TValue>`
- `Bool`

Nullables are also supported.
